# Landing Page em React

Este projeto tem como objetivo demonstrar, de forma simples, o uso do React na construcao de uma landing page. O tema escolhido foi uma empresa de irrigacao generica, apenas como exemplo visual e estrutural.

## O que foi utilizado

- React: para montar a interface em componentes e organizar melhor a estrutura da pagina.
- Vite: para criar um ambiente de desenvolvimento rapido e leve.
- CSS puro: para estilizar a landing page sem depender de bibliotecas externas.

## Por que essa base

Essa estrutura foi escolhida para mostrar um fluxo comum em projetos React:

- renderizacao de uma interface unica
- separacao entre estrutura e estilo
- facilidade para editar textos, secoes e identidade visual
